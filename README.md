# n8n Automation Projects

A collection of automation workflows built using [n8n](https://n8n.io/), a no-code/low-code workflow automation tool. These projects demonstrate practical business process automation and AI agent design.

---

## 1. Automated Job Application Workflow

**File:** `job-application-automation.json`

An end-to-end recruitment automation. When a candidate submits a job application form, the workflow automatically logs their details into a Google Sheet and sends a personalized confirmation email — eliminating manual tracking and follow-up.

**Flow:**
`Form Submission → Append Row in Google Sheet → Send Confirmation Email (Gmail)`

**Tech used:**
- n8n (workflow orchestration)
- Google Sheets API
- Gmail API
